Seam Carving

Overview

In this project, I implemented the basic algorithm presented by Shai Avidan and Ariel Shamir in Seam Carving for Content-Aware Image Resizing (SIGGRAPH). Whereas typical cropping methods fail to consider content when resizing an image, this algorithm first removes the seams (connected paths of pixels running either vertically or horizontally across the image) which it deems of least importance. Below is a GIF of my code in action (ironically, the website I used (ezgif.com) automatically resized each frame to fit the dimensions of the first – regardless, it illustrates the seam selection).

In implementing Seam Carving, I first determined the “importance” of each pixel, employing the image’s gradient magnitude as my energy function. Because the gradient magnitude uses Sobel filters to approximate the image’s partial derivatives, low cost pixels represent areas of little change. As such, this energy function prioritizes the removal of seams in homogeneous sections of the image. Below is an example of an image’s gradient magnitude.

Once every pixel was assigned a cost, I used backtracking to create a cumulative energy matrix, determining the continuous path (seam) which requires the least energy to traverse. This seam was then used to create a Boolean mask which, when applied to the original image, allowed for the pixels’ removal. By repeating this process on the resized result, one can iteratively reduce an image’s width or height by a single pixel until the desired dimension is achieved. I ran the algorithm on various movie stills (taken from film-grab.com). The results are featured below.

Because this process can be rather slow, I further divided the function into a precompute section and an online section. To implement the former, I resized the image until it was a single pixel in either width or height, saving each seams’ mask into a matrix before removal. With this matrix, the online method can reduce the image to any new dimension (in a matter of seconds) by simply applying the necessary number of masks in order.
